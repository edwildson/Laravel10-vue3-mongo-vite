# LARAVEL 10 + VUE 3 + MONGO + VITE + CUIDA

Repositório para se utilizar como base para criação de aplicações com a stack LARAVUE atualizada.

## Installation

Para instalar execute os comandos abaixo.

```bash
composer i
npm i
php artisan key:g
php artisan serve
npm run dev
```
